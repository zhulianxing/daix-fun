💡 无推广者时的分配

如果购买时没有推广者（直接购买），推广者 25% 归入创作者，即创作者获得 85%

代码实现

uint256 public constant DEV_SHARE = 6000;      // 60% 创作者
uint256 public constant AGENT_SHARE = 2500;    // 25% 推广者
uint256 public constant PLATFORM_SHARE = 1000; // 10% 生态基金
uint256 public constant GAS_SHARE = 500;       // 5%  网络结算

// 分账逻辑
uint256 devAmount = (price * DEV_SHARE) / 10000;
uint256 agentAmount = (agent != address(0)) ? (price * AGENT_SHARE) / 10000 : 0;
uint256 platformAmount = (price * PLATFORM_SHARE) / 10000;
uint256 gasAmount = (price * GAS_SHARE) / 10000;
uint256 devTotal = devAmount + agentAmount;  // 无推广者时归开发者

usdc.transfer(app.developer, devTotal);
usdc.transfer(platformWallet, platformAmount);
usdc.transfer(gasWallet, gasAmount);
if (agent != address(0)) usdc.transfer(agent, agentAmount);

🛒 购买流程

从钱包连接到获得 NFT 数字凭证，全程链上操作

1
🔗 连接钱包

浏览器安装 MetaMask，选择 Polygon 网络，点击「连接钱包」

2
📱 选择应用

浏览应用商店，点击心仪应用的「获取数字凭证」按钮

3
✅ 授权 USDC

钱包弹出授权交易，允许 LicenseNFT 合约使用对应金额的 USDC

4
🔑 铸造 NFT

第二笔交易 mintLicense，智能合约自动分账，NFT 即时到账

交易步骤详解

// Step 1: 授权 USDC
usdc.approve(LicenseNFT, price);
// → 用户钱包确认

// Step 2: 铸造数字凭证
licenseNFT.mintLicense(appId, agentAddress);
// → 自动分账：创作者 60% | 推广者 25% | 生态 10% | 网络 5%
// → 铸造 ERC-721 NFT 到用户钱包

🚀 快速起步

1 分钟了解如何开始使用 DAIX.fun

👤 作为用户

  1. 安装 MetaMask 浏览器钱包
  2. 向钱包充值 Polygon 主网 USDC 和少量 POL（作为 Gas 费）
  3. 打开 DAIX.fun
  4. 浏览应用，点击「获取数字凭证」完成购买
  5. 在「我的数字凭证」中查看和管理持有的 NFT 凭证

👨‍💻 作为创作者

  1. 连接钱包，进入「创作者中心」
  2. 注册创作者身份（仅需名称，链上永久有效）
  3. 发布应用（填写名称、包名、价格等信息）
  4. 选择 APK 文件，自动上传至 IPFS 去中心化存储
  5. APK 的 IPFS CID 写入链上 apkURI 字段，永久可追溯
  6. 用户购买后自动获得 60% 收益分成
📦 IPFS 去中心化存储
APK 文件通过 Pinata API 上传至 IPFS 网络，获得唯一 CID。
下载链接从链上 apkURI 字段读取，解析为 IPFS 公共网关 URL。
创作者无需自建服务器，APP 文件全球可访问、抗审查。

📣 作为推广者

  1. 连接钱包，进入「推广者中心」
  2. 注册推广者身份（设置名称和唯一推广码）
  3. 生成推广链接：https://daix.fun/?agent=你的钱包地址
  4. 用户通过你的链接购买，你自动获得 25% 佣金
